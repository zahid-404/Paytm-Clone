## Features

1. **Secure Authentication:**
   - Create an account and log in securely using JWT. üîê

2. **Random Balance Assignment:**
   - Upon login, users are assigned a random balance for transactions. üí∞

3. **User Search and Money Transfer:**
   - Easily search for other users and send money effortlessly. üí∏

4. **Responsive UI with Modern Components:**
   - UI is fully responsive and built using React, Tailwind CSS, and NextUI components, ensuring a seamless experience across devices. üì±‚ú®

5. **ACID Operations for Transaction Integrity:**
   - Utilized Mongoose for efficient database management and ACID operations, ensuring transactions are either fulfilled or rejected completely. üîÑ

## Installation

1. Clone this repository: `https://github.com/zahid-404/Paytm-Clone`
2. Navigate to the project directory: `cd Paytm-Clone`
3. Navigate to the Backend: `cd frontend`
4. Install dependencies for frontend:
   ```bash
   npm install
   ```
5. Navigate to the Backend: `cd backend`
6. Install dependencies for frontend:
   ```bash
   npm install
   ```

## Usage

1. Start the frontend:

```bash
  npm run dev
```

1. Start the Backend:

```bash
  npm run dev
```
